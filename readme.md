# 🎬 Image to Video Exporter

A powerful browser-based tool that transforms a single image into a smooth cinematic zoom animation. Export your animation as high-quality WebM video or GIF directly from the browser — no software installation needed.

## ✨ Features

- 🖼️ Upload JPG, PNG, WebP images
- 🔍 Smooth Zoom In / Zoom Out animation
- ✂️ Advanced crop selection
  - Full Crop
  - Free Crop
  - Square Crop
- 🎞️ Export as:
  - WebM Video
  - GIF
- ⚡ Adjustable:
  - Duration
  - FPS
  - Zoom Strength
  - Resolution
  - Video Quality
- 📱 Supports:
  - Full HD
  - 2K
  - 4K
  - Shorts/Reels format
  - Square format
- 🎨 Modern Glassmorphism UI
- 🚀 Fully client-side (Runs locally in browser)

---

## 🛠️ Technologies Used

- HTML5
- Tailwind CSS
- Vanilla JavaScript
- GIF.js
- Canvas API
- MediaRecorder API
